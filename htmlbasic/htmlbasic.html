Introducció a HTML

Què és HTML?

  • HTML (HyperText Markup Language) és el llenguatge de marques per crear pàgines web.

  • Defineix l'estructura de la pàgina amb etiquetes que el navegador interpreta per mostrar contingut.

Breu Història

  • Creat per Tim Berners-Lee el 1991.

  • HTML 2.0 (1995): Primer estàndard oficial.

  • HTML5 (2014): Última versió amb noves funcionalitats com suport per a vídeo, àudio i gràfics.

Estructura Bàsica d'un Document HTML

Estructura General

  • Un document HTML es basa en una estructura jeràrquica de etiquetes que defineixen les diferents parts d'una pàgina.
  • Cada pàgina HTML segueix una estructura similar, començant pel !DOCTYPE, seguit de les etiquetes html, head, body.

Etiquetes Bàsiques

  • title

    • Defineix el títol que apareix a la pestanya del navegador o als resultats dels cercadors.

  • h1 a h6

    • Utilitzats per definir títols i subtítols a la pàgina web.

  • p

    • S’utilitza per definir blocs de text o paràgrafs.

  • br

    • Força el text a començar en una nova línia.

  • & nbsp;

    • Inserta un espai en blanc que no es trenca entre línies.
